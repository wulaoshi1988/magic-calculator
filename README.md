## 🎯 快速测试

### 方法1：直接打开文件
1. 在文件管理器中找到 `magic-calculator/index.html`
2. 双击用浏览器打开

### 方法2：本地服务器
```bash
cd magic-calculator
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🐛 常见问题

### 双击不触发
- 确保点击的是页面空白区域（非按钮）

### 时间计算错误
- 检查浏览器时区设置
- 确保24小时制正确显示

### 动画卡顿
- 尝试关闭其他标签页
- 检查浏览器性能

### 触摸无响应
- 确保触摸区域正确
- 检查移动端触摸事件

## 🔧 调试技巧

### 查看控制台
```javascript
// 在浏览器控制台运行
console.log(window.magicCalculator);
console.log(window.calculator);
```

### 强制测试魔术模式
```javascript
// 在控制台直接触发
window.magicCalculator.startMagicMode();
```

### 检查时间计算
```javascript
// 验证算法
const time = window.magicCalculator.getCurrentTime();
console.log('当前时间数字:', time);
```
